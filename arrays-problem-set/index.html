isUniform()

Write a function isUniform() which takes an array as an argument and returns if all elements in the array are identical

Ex:

isUniform([1,2,3,4]); //true

isUniform([2,1,1,1]); //false

isUniform(["a","b","p"]); //false

isUniform(["b","b","b"]); //true
